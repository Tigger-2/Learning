# Software Engineering Fundamentals Learning Programme

## Core Concepts (Weeks 1-4)

### Topics & Resources

- **Programming Paradigms**: Object-oriented, functional, procedural programming
  - [Wikipedia - Programming Paradigm](https://en.wikipedia.org/wiki/Programming_paradigm)
  - [GeeksforGeeks - Programming Paradigms](https://www.geeksforgeeks.org/programming-paradigms-in-software-engineering/)
  - [Functional Programming in Python - Real Python](https://realpython.com/functional-programming-python/)

- **Design Principles**: SOLID principles (Single Responsibility, Open/Closed, Liskov, Interface Segregation, Dependency Inversion)
  - [SOLID Principles - Wikipedia](https://en.wikipedia.org/wiki/SOLID)
  - [SOLID Principles Explained - DigitalOcean](https://www.digitalocean.com/community/tutorials/s-o-l-i-d-principles-explained-in-python)
  - [Real Python - SOLID Principles](https://realpython.com/solid-principles/)
  - [Baeldung - SOLID Principles](https://www.baeldung.com/solid-principles)

- **Code Quality**: Writing clean code, readability, maintainability
  - [Clean Code - Robert C. Martin](https://www.oreilly.com/library/view/clean-code-a/9780136083238/)
  - [Code Smell - Refactoring Guru](https://refactoring.guru/refactoring/smells)
  - [Real Python - How to Write Good Code](https://realpython.com/python-code-quality/)
  - [PEP 8 - Style Guide for Python Code](https://pep8.org/)

- **Version Control**: Git fundamentals, branching, merging, collaboration
  - [Git Official Documentation](https://git-scm.com/doc)
  - [Pro Git Book](https://git-scm.com/book/en/v2)
  - [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials)
  - [GitHub - Hello World](https://guides.github.com/activities/hello-world/)
  - [GitHub Learning Lab](https://lab.github.com/)

### Practice Projects
- Refactor existing code, follow best practices
- [Exercism - Code Practice](https://exercism.io/)
- [LeetCode - Code Interview Preparation](https://leetcode.com/)
