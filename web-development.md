# Web Development Learning Programme

## Frontend Fundamentals (Weeks 1-4)

### Topics & Resources

- **HTML5 Basics**: Semantic HTML, forms, media elements
  - [HTML Official Documentation - W3C](https://html.spec.whatwg.org/)
  - [MDN Web Docs - HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
  - [W3Schools - HTML Tutorial](https://www.w3schools.com/html/)
  - [HTML5 Semantic Elements](https://developer.mozilla.org/en-US/docs/Web/HTML/Element)

- **CSS Styling**: Selectors, box model, flexbox, grid layouts
  - [MDN Web Docs - CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
  - [W3Schools - CSS Tutorial](https://www.w3schools.com/css/)
  - [CSS-Tricks - Flexbox Guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)
  - [CSS-Tricks - Grid Guide](https://css-tricks.com/snippets/css/complete-guide-grid/)
  - [CSS Selectors Reference](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors)

- **Responsive Design**: Media queries, mobile-first approach
  - [MDN - Responsive Design](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Responsive_Design)
  - [Google Mobile-Friendly Test](https://search.google.com/test/mobile-friendly)
  - [Responsive Web Design Basics - Google](https://developers.google.com/web/fundamentals/design-and-ux/responsive)
  - [Media Queries Reference](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries)

### Practice Projects
- Build static websites (portfolio, landing page, blog)
- [Frontend Mentor - Design Challenges](https://www.frontendmentor.io/)
- [CodePen - HTML/CSS Showcase](https://codepen.io/)

## JavaScript Essentials (Weeks 5-8)

### Topics & Resources

- **JavaScript Fundamentals**: Variables, functions, scope, closures
  - [MDN Web Docs - JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
  - [W3Schools - JavaScript Tutorial](https://www.w3schools.com/js/)
  - [JavaScript.info - Modern JavaScript Course](https://javascript.info/)
  - [Eloquent JavaScript Book](https://eloquentjavascript.net/)

- **DOM Manipulation**: Selecting, modifying, and handling events
  - [MDN - DOM Manipulation](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model)
  - [MDN - Events](https://developer.mozilla.org/en-US/docs/Web/Events)
  - [W3Schools - DOM Manipulation](https://www.w3schools.com/js/js_dom.asp)
  - [Query Selector API Reference](https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector)

- **Async JavaScript**: Promises, async/await, fetch API
  - [MDN - Promises](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)
  - [MDN - Async/Await](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/async_function)
  - [MDN - Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)
  - [JavaScript.info - Promises](https://javascript.info/promise-basics)

- **ES6+ Features**: Arrow functions, destructuring, spread operator, modules
  - [MDN - ES6 Features](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Modules)
  - [ES6 Cheatsheet - GitHub](https://github.com/DrkSephy/es6-cheatsheet)
  - [Arrow Functions - MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/Arrow_functions)
  - [Destructuring - MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment)

### Practice Projects
- Build interactive web pages, form validation, API integration
- [JavaScript 30 - Vanilla JS Projects](https://javascript30.com/)
- [Frontend Mentor - JavaScript Projects](https://www.frontendmentor.io/)

## Backend Basics (Weeks 13-16)

### Topics & Resources

- **Node.js & Express**: Server setup, routing, middleware
  - [Node.js Official Documentation](https://nodejs.org/en/docs/)
  - [Express.js Official Documentation](https://expressjs.com/)

- **Databases**: SQL (PostgreSQL) and NoSQL (MongoDB) fundamentals
  - [PostgreSQL Official Documentation](https://www.postgresql.org/docs/)
  - [MongoDB Official Documentation](https://docs.mongodb.com/)
  - [SQL Tutorial - W3Schools](https://www.w3schools.com/sql/)

- **REST APIs**: Building RESTful APIs, HTTP methods
  - [RESTful API Tutorial](https://restfulapi.net/)

### Practice Projects
- Build complete RESTful APIs with authentication
