# Web Development Learning Programme

## Frontend Fundamentals (Weeks 1-4)
- **HTML5 Basics**: Semantic HTML, forms, media elements
- **CSS Styling**: Selectors, box model, flexbox, grid layouts
- **Responsive Design**: Media queries, mobile-first approach
- **Practice**: Build static websites (portfolio, landing page, blog)

## JavaScript Essentials (Weeks 5-8)
- **JavaScript Fundamentals**: Variables, functions, scope, closures
- **DOM Manipulation**: Selecting, modifying, and handling events
- **Async JavaScript**: Promises, async/await, fetch API
- **ES6+ Features**: Arrow functions, destructuring, spread operator, modules
- **Practice**: Interactive web pages, form validation, API integration

## Frontend Frameworks (Weeks 9-12)
- **React Basics**: Components, JSX, props, state, hooks
- **State Management**: useState, useEffect, custom hooks, context API
- **Routing & Navigation**: React Router
- **Practice**: Build single-page applications (note app, weather app)

## Backend Basics (Weeks 13-16)
- **Node.js & Express**: Server setup, routing, middleware
- **Databases**: SQL (PostgreSQL) and NoSQL (MongoDB) fundamentals
- **REST APIs**: Building RESTful APIs, HTTP methods
- **Authentication**: User authentication, JWT tokens

## Capstone Project
- Build a full-stack application: Frontend + Backend + Database
- Examples: Social media app, e-commerce platform, project management tool
