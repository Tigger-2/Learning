# Learning Programmes

Welcome to the Learning repository! This collection contains structured learning programmes for different areas of software development and technology.

## Available Learning Programmes

### 1. [Python Learning Programme](python-learning.md)
A comprehensive 12-week programme covering Python fundamentals, intermediate concepts, and advanced topics. Perfect for beginners looking to master Python programming.
- Beginner: Variables, control flow, functions, data structures
- Intermediate: OOP, file I/O, modules
- Advanced: Decorators, design patterns, testing

### 2. [Web Development Learning Programme](web-development.md)
A 16-week full-stack development journey from HTML/CSS basics to building complete applications.
- Frontend: HTML5, CSS, JavaScript, React
- Backend: Node.js, Express, Databases
- Full-stack projects and deployment

### 3. [Data Science Learning Programme](data-science.md)
A 16-week programme covering data manipulation, visualization, machine learning, and deep learning.
- Foundations: Math, Python, data manipulation
- ML: Supervised/unsupervised learning, model evaluation
- Advanced: Deep learning, time series, feature engineering

### 4. [Software Engineering Fundamentals Learning Programme](software-engineering.md)
A 16-week course on professional software development practices and principles.
- Architecture: Design patterns, system design
- Testing: Unit tests, integration tests, QA
- DevOps: CI/CD, containerization, deployment, security

## How to Use This Repository

1. Choose a learning programme based on your interests and goals
2. Follow the structured progression from fundamentals to advanced topics
3. Complete practice projects at each stage
4. Build capstone projects to demonstrate mastery
