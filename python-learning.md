# Python Learning Programme

## Beginner Level (Weeks 1-4)

### Topics & Resources

- **Variables & Data Types**: Understanding strings, integers, floats, booleans
  - [Python Official Documentation - Built-in Types](https://docs.python.org/3/library/stdtypes.html)
  - [Real Python - Variables and Data Types](https://realpython.com/python-data-types/)
  - [W3Schools - Python Data Types](https://www.w3schools.com/python/python_datatypes.asp)

- **Control Flow**: if/else statements, loops (for, while)
  - [Python Official Documentation - Control Flow](https://docs.python.org/3/tutorial/controlflow.html)
  - [Real Python - Python Loops](https://realpython.com/loops/)
  - [Codecademy - Control Flow Lessons](https://www.codecademy.com/learn/learn-python-3)

- **Functions**: Defining functions, parameters, return values
  - [Python Official Documentation - Functions](https://docs.python.org/3/tutorial/controlflow.html#defining-functions)
  - [Real Python - Python Functions](https://realpython.com/defining-your-own-python-function/)
  - [GeeksforGeeks - Functions in Python](https://www.geeksforgeeks.org/python-functions/)

- **Basic Data Structures**: Lists, tuples, dictionaries, sets
  - [Python Official Documentation - Data Structures](https://docs.python.org/3/tutorial/datastructures.html)
  - [Real Python - Lists and Tuples](https://realpython.com/python-lists-tuples/)
  - [Real Python - Dictionaries](https://realpython.com/python-dicts/)

### Practice Projects
- Build simple programs (calculator, to-do list, number guessing game)
- [GitHub - Real Python Code Examples](https://github.com/realpython)

## Intermediate Level (Weeks 5-8)

### Topics & Resources

- **File I/O**: Reading and writing files
  - [Python Official Documentation - File I/O](https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files)
  - [Real Python - Reading and Writing Files in Python](https://realpython.com/read-write-files-python/)
  - [GeeksforGeeks - File Handling in Python](https://www.geeksforgeeks.org/file-handling-python/)

- **Error Handling**: Try/except blocks, exception handling
  - [Python Official Documentation - Exceptions](https://docs.python.org/3/tutorial/errors.html)
  - [Real Python - Exception Handling in Python](https://realpython.com/python-exceptions/)
  - [Real Python - The try-except Block](https://realpython.com/python-try-except/)

- **Object-Oriented Programming**: Classes, objects, inheritance, polymorphism
  - [Python Official Documentation - Classes](https://docs.python.org/3/tutorial/classes.html)
  - [Real Python - Object-Oriented Programming (OOP) in Python 3](https://realpython.com/python3-object-oriented-programming/)
  - [Real Python - Inheritance and Composition in Python](https://realpython.com/inheritance-composition-python/)

- **Modules & Packages**: Import statements, using built-in and third-party libraries
  - [Python Official Documentation - Modules](https://docs.python.org/3/tutorial/modules.html)
  - [Real Python - Python Modules and Packages](https://realpython.com/python-modules-packages/)
  - [PyPI - The Python Package Index](https://pypi.org/)

### Practice Projects
- Build a project management tool, file organizer application
- [Real Python - Projects](https://realpython.com/tutorials/projects/)

## Advanced Level (Weeks 9-12)

### Topics & Resources

- **Decorators & Generators**: Advanced function concepts
  - [Python Official Documentation - Decorators](https://docs.python.org/3/glossary.html#term-decorator)
  - [Real Python - Primer on Python Decorators](https://realpython.com/primer-on-python-decorators/)
  - [Real Python - Generators and yield in Python](https://realpython.com/introduction-to-python-generators/)

- **Context Managers**: Using with statements
  - [Python Official Documentation - Context Managers](https://docs.python.org/3/library/contextlib.html)
  - [Real Python - Context Managers](https://realpython.com/context-managers-and-with-statements-in-python/)

- **Design Patterns**: Singleton, Factory, Observer patterns
  - [Real Python - Design Patterns in Python](https://realpython.com/python-design-patterns/)
  - [Refactoring Guru - Design Patterns](https://refactoring.guru/design-patterns/python)
  - [GeeksforGeeks - Design Patterns in Python](https://www.geeksforgeeks.org/design-patterns-in-python/)

- **Testing**: Unit tests with unittest/pytest
  - [Python Official Documentation - unittest](https://docs.python.org/3/library/unittest.html)
  - [Real Python - Unit Testing in Python](https://realpython.com/python-testing/)
  - [pytest Documentation](https://docs.pytest.org/)
  - [Real Python - Getting Started With Testing](https://realpython.com/python-testing/)

### Practice Projects
- Build a web scraper, automation script, or CLI tool
- [Real Python - Web Scraping Projects](https://realpython.com/web-scraping-with-beautiful-soup/)
- [Create Your Own Command Line Tool - Real Python](https://realpython.com/command-line-interfaces-with-click/)

## Capstone Project

Create a full-featured application combining all learned concepts.

### Project Ideas
- **Task Manager with Database**: SQLite/PostgreSQL integration, GUI with tkinter or PyQt
- **Personal Finance Tracker**: Data analysis, visualization, budget management
- **Data Analysis Tool**: Process and visualize datasets with Pandas and Matplotlib
