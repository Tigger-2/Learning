# Python Learning Programme

## Beginner Level (Weeks 1-4)
- **Variables & Data Types**: Understanding strings, integers, floats, booleans
- **Control Flow**: if/else statements, loops (for, while)
- **Functions**: Defining functions, parameters, return values
- **Basic Data Structures**: Lists, tuples, dictionaries, sets
- **Practice**: Build simple programs (calculator, to-do list, number guessing game)

## Intermediate Level (Weeks 5-8)
- **File I/O**: Reading and writing files
- **Error Handling**: Try/except blocks, exception handling
- **Object-Oriented Programming**: Classes, objects, inheritance, polymorphism
- **Modules & Packages**: Import statements, using built-in and third-party libraries
- **Practice**: Build a project management tool, file organizer application

## Advanced Level (Weeks 9-12)
- **Decorators & Generators**: Advanced function concepts
- **Context Managers**: Using with statements
- **Design Patterns**: Singleton, Factory, Observer patterns
- **Testing**: Unit tests with unittest/pytest
- **Practice**: Build a web scraper, automation script, or CLI tool

## Capstone Project
- Create a full-featured application combining all learned concepts
- Examples: Task manager with database, personal finance tracker, data analysis tool
