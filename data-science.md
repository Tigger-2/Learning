# Data Science Learning Programme

## Foundations (Weeks 1-4)
- **Mathematics**: Linear algebra, calculus, probability, statistics
- **Python for Data Science**: NumPy, Pandas fundamentals
- **Data Manipulation**: Cleaning, transforming, aggregating datasets
- **Practice**: Analyze real-world datasets, data cleanup exercises

## Data Visualization & Exploration (Weeks 5-8)
- **Visualization Libraries**: Matplotlib, Seaborn, Plotly
- **Exploratory Data Analysis (EDA)**: Identifying patterns, outliers, distributions
- **Statistical Analysis**: Hypothesis testing, correlation analysis
- **Practice**: Create dashboards, visualize complex datasets

## Machine Learning Fundamentals (Weeks 9-12)
- **Supervised Learning**: Linear regression, logistic regression, decision trees
- **Unsupervised Learning**: K-means clustering, hierarchical clustering, PCA
- **Model Evaluation**: Cross-validation, metrics (accuracy, precision, recall, F1)
- **scikit-learn**: Training and evaluating models
- **Practice**: Build predictive models, classification tasks

## Advanced Topics (Weeks 13-16)
- **Deep Learning**: Neural networks, TensorFlow/PyTorch basics
- **Time Series Analysis**: ARIMA, forecasting techniques
- **Feature Engineering**: Creating meaningful features, dimensionality reduction
- **Hyperparameter Tuning**: Grid search, random search

## Capstone Project
- End-to-end data science project from problem definition to deployment
- Examples: Predictive analytics, customer segmentation, recommendation system, time series forecasting
